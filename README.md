<h2>ğŸ“˜ About Linear Regression</h2>
<p>
Linear Regression is a supervised machine learning algorithm used to model the relationship
between a dependent variable and one or more independent variables. The model fits a straight
line that minimizes the error between predicted and actual values.
</p>

<p>
In this project, Linear Regression is used to understand how different vehicle features such as
engine power, weight, and acceleration affect fuel usage.
</p>
